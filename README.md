## GoogleMeetの字幕をもっと認知しやすくするための拡張機能。  

このアプリはGoogleMeetの字幕機能をもっとわかりやすく表示させて、人間が会話を理解する能力を高める拡張機能です。  
人間の知覚による割合は、視覚83%、聴覚11%、嗅覚3.5%、触覚1.5%、味覚1%とされています。  
音声情報はノイズが乗りやすく、認知負荷も高いため、その負荷を視覚に分散することで認知負荷を軽減することができます。
